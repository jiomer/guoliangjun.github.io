Mysql时间与字符串之间相互转换

发表于 2019-08-20 更新于 2021-06-10 分类于 学习 阅读次数：

因为mysql的时间转换与oracle是有点区别，再上一家公司，是以oracle为主，重新用mysql需要在习惯一下，就记录一下呗~

时间转字符串：

DATE_FORMAT(日期，格式字符串) SELECT DATE_FORMAT(NOW(), ‘%Y-%m-%d %H:%i:%s’);–now()为当期时间 –结果：2019-08-20 20:40:08

字符串转时间：

STR_TO_DATE(字符串，日志格式) SELECT STR_TO_DATE(‘2019-08-20 20:40:08’, ‘%Y-%m-%d %H:%i’); –结果：2019-08-20 20:40:00

时间转时间戳：

select unix_timestamp(now());–now()为当期时间 –结果：1566304856

字符串转时间戳：

select unix_timestamp(‘2019-08-20’); –结果：1566230400

时间戳转字符串：

select from_unixtime(1566230400,’%Y-%m-%d’); –结果：2019-08-20
